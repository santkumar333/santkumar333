### 👋 About

I lead **AI-driven test automation and GenAI quality engineering** — designing **agentic AI pipelines** (Claude, LangChain, LangGraph, MCP) that auto-generate test cases from requirements, keep requirement-to-test coverage continuously auditable, and cut manual QA effort dramatically. At **Mercedes-Benz R&D India** I build these systems for vehicle connectivity and Android infotainment, and mentor QA + dev teams to adopt agentic SDLC workflows.

I'm the person who takes an LLM from "cool demo" to **auditable, ASPICE-ready quality tooling** in a safety-critical domain.

### 🚀 Impact highlights

- 🧩 **LLM requirements-traceability** across **1,000+ requirements** → **80% less** manual traceability effort, with real-time coverage + ASPICE-compliant audit trails.
- ⚙️ **GenAI agentic pipelines** (Claude + LangChain + LangGraph + MCP) that **auto-generate test cases** from requirements.
- 🤖 **AI-powered ASPICE agentic system** — agents track merge requests, detect architecture changes, and auto-draft documentation for architect review.
- ⚡ **Autonomous AI regression** with predictive defect detection → **60% faster** execution cycles via intelligent test prioritization.
